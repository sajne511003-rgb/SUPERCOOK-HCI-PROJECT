

# SUPERCOOK-HCI-PROJECT

## 1. Project Overview
SuperCook is a fully functional smartphone application developed to address the **cognitive friction** experienced by beginner cooks when dealing with complex cooking terminology or missing ingredients. The project follows the **iterative Human-Computer Interaction (HCI) design lifecycle**, ensuring an inclusive, user-friendly, and resilient digital solution tailored to novice users.

---

## 2. Fully Functional MVP App
- **Platform Notes:**  
  This application has been optimized and tested for **Android devices**.

---

## 3. MVP Must-Have Features
The following features were prioritized as *Must-Haves* in Task 1 and have been fully implemented in this high-fidelity prototype:

- **Interactive Recipe Finder**  
  Enables users to discover beginner-friendly recipes easily.

- **Instant Ingredient Substitution**  
  A “Swap” feature that suggests alternative ingredients when items are missing, justified using **Fitts’ Law** to ensure efficiency and ease of interaction.

- **Terminology Help**  
  Integrated explanations for cooking terms to support the **Simple and Intuitive Use** principle of **Universal Design**.
